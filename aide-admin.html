Bienvenue ! Cette page rassemble toutes les procédures pour gérer le site BCCO depuis l'admin : actualités, effectifs, galerie photos, classement, et plus encore. Pas besoin de connaissances techniques — tout se fait depuis ton navigateur.

🔐 Se connecter à l'admin

L'admin est accessible via le lien "Admin" en bas à droite du site, dans le footer (petit lien gris).

1
Ouvre l'admin URL directe : chamblybadminton.fr/admin-bcco-fe732ff3.php
2
Saisis ton identifiant et ton mot de passe Ils t'ont été communiqués par Théo. Pas de "mot de passe oublié" automatique — contacte un super-admin.
3
Tu es connecté La page de l'admin s'ouvre. Ta session reste active tant que tu ne fermes pas l'onglet du navigateur.
⚠️ Si tu essaies plusieurs fois avec de mauvais identifiants, l'accès se bloque pendant 30 minutes. Patiente ou contacte un super-admin.

🔑 Configurer ton token GitHub (1ère fois)

Pour que l'admin puisse écrire sur le site (publier des actus, modifier les effectifs…), il a besoin d'un "token" personnel GitHub. C'est ta clé personnelle, à faire UNE SEULE FOIS.

1
Crée un compte GitHub si tu n'en as pas Va sur github.com → Sign up. Gratuit, 2 minutes.
2
Demande à Théo de t'ajouter comme collaborateur Il t'envoie une invitation sur GitHub que tu dois accepter.
3
Génère ton token Va sur cette page (le scope "repo" est déjà coché). Donne un nom (ex: "BCCO Admin"), choisis une expiration (90 jours recommandé), clique "Generate token".
4
Copie le token (il commence par ghp_) ⚠️ Tu ne pourras plus le revoir après. Si tu le perds, tu devras en générer un nouveau.
5
Colle-le dans l'admin La première fois que tu fais une modification, une fenêtre te demande le token. Colle-le, clique "Connecter".
💡 Le token est stocké uniquement dans ton navigateur (sessionStorage). Il disparaît quand tu fermes l'onglet. À chaque nouvelle session admin, il te sera redemandé.

📰 Publier une actualité

Les actualités s'affichent sur la home du site, dans la section "Prochains rendez-vous". Tu peux en avoir 4-8 actives en même temps.

1
Va dans la section "Actualités du site" Tu vois la liste des actus en cours sous forme de cartes.
2
Clique sur le bouton vert "+ Nouvelle actualité" Un formulaire s'ouvre dans une fenêtre.
3
Remplis les champs Date de publication : aujourd'hui (sert au tri, invisible sur le site).
Date de l'événement : texte affiché sur la carte (ex: "Samedi 28 mars · 16h00").
Titre, Résumé : obligatoires.
Image : glisser-déposer un JPEG/PNG (sera redimensionnée à 1600px max automatiquement) OU coller une URL d'image externe.
Catégorie : Match (vert), Tournoi (orange), Événement (bleu), ou aucune.
Lien externe : facultatif (ajoute un bouton "En savoir plus").
4
Coche "Publier cette actualité" Sinon elle reste en brouillon (badge gris "Brouillon" dans l'admin, invisible sur le site).
5
Clique sur "Aperçu" pour vérifier le rendu Tu vois exactement comment ta carte apparaîtra sur la home.
6
Clique "Publier" L'actu est commitée sur GitHub et apparaît sur le site dans 1-2 minutes (le temps de propagation).
💡 Tu peux modifier une actu en cliquant sur "Modifier" sur sa carte, ou la supprimer avec "Supprimer".

⏰ Programmer une actu pour plus tard

Tu peux préparer une annonce à l'avance et la faire apparaître automatiquement à une date future.

1
Crée l'actu normalement Comme dans la section précédente.
2
Mets une "Date de publication" dans le futur Exemple : tu prépares aujourd'hui l'annonce d'un match du 15 juin. Mets 2026-06-15 comme date de publication.
3
Coche "Publier" et clique "Publier" L'actu est commitée mais ne sera PAS visible sur le site avant la date prévue.
4
Dans l'admin, badge orange "⏰ Programmée · 15 juin 2026" La carte a une bordure orange pour bien la distinguer des actus déjà visibles.
💡 Le jour J, l'actu apparaît automatiquement sur le site (pas besoin d'action de ta part).

👥 Gérer les licenciés

La section "Effectifs licenciés" liste tous les joueurs avec leur équipe, classement (simple/double/mixte), et statut actif/inactif.

Ajouter un licencié

  1. Clique sur le bouton vert "+ Nouveau licencié".
  2. Saisis l'équipe (top12, n2, r2, d3, icd, oise, jeunes…). L'auto-complétion te propose les codes existants.
  3. Nom, prénom obligatoires.
  4. Classements simple/double/mixte : sélectionne dans la liste (N1, N2, R4, D9, P10…).
  5. Coche "Capitaine" si c'est le capitaine de l'équipe.
  6. Coche "Actif" pour qu'il/elle apparaisse sur le site.
  7. Clique "Enregistrer".

Modifier ou désactiver un licencié

  • Modifier : clique sur le bouton "Éditer" de la ligne.
  • Activer/désactiver : clique sur le toggle vert/gris dans la colonne "Actif".
  • Supprimer : bouton 🗑️ (avec confirmation).

Rechercher et filtrer

  • Filtre déroulant par équipe (Top 12, Nationale 2, Régionale 2…) avec le compteur entre parenthèses.
  • Recherche par nom/prénom dans la barre de recherche.
  • Tri en cliquant sur les en-têtes de colonnes (Équipe, Nom, Prénom).
💡 Sur mobile, certaines colonnes (Simple/Double/Mixte/Capitaine) sont cachées pour la lisibilité. Tape sur "Éditer" pour les voir.

🖼️ Ajouter des photos à la galerie

La galerie photos est visible sur galerie.html. Tu peux y ajouter des photos en quelques clics.

1
Va dans la section "Galerie photos" Tu vois les photos actuelles sous forme de petites vignettes.
2
Glisse tes photos dans la zone de dépôt Plusieurs à la fois (JPEG, PNG ou WebP). Elles sont redimensionnées automatiquement à 1800px max.
3
Édite les infos de chaque photo Légende (texte qui apparaît au survol), Taille (Normal, Large, Haute, XL 2x2), case "Publiée" pour la rendre visible.
4
Réorganise avec les flèches ↑ ↓ Pour changer l'ordre d'affichage dans la galerie.
5
Clique "Enregistrer" sur chaque carte modifiée Les changements sont commités sur GitHub à chaque action.
💡 Pour gagner du poids, utilise le convertisseur WebP avant d'uploader → tes photos seront 60-70% plus légères, donc le site se charge plus vite.

📥 Importer un CSV (classement, palmarès…)

Pour les rencontres, classement Top 12 et palmarès joueurs, il est plus rapide de gérer dans Numbers/Excel puis d'importer le fichier.

1
Édite ton fichier dans Numbers (ou Excel) Garde les mêmes colonnes que l'original.
2
Exporte en CSV Numbers : Fichier → Exporter vers → CSV…
3
Va dans la section "Import CSV" Dans la liste déroulante, choisis le fichier à remplacer (Rencontres, Classement, ou Palmarès).
4
Glisse ton CSV dans la zone de dépôt Le module détecte automatiquement le séparateur (Numbers FR utilise ;) et le convertit.
5
Vérifie l'aperçu Tu vois "Ancien : X lignes → Nouveau : Y lignes" + warnings éventuels.
6
Clique "Remplacer" L'ancien fichier est écrasé. Le site est mis à jour dans 1-2 minutes.
⚠️ L'import ne marche PAS pour Actualités, Effectifs et Galerie — utilise leurs sections dédiées pour éviter d'écraser des modifs.

🗜️ Optimiser des images (JPEG/PNG → WebP)

Le convertisseur intégré te permet de réduire le poids des images jusqu'à 70% sans perte visible, pour que le site charge plus vite.

1
Va dans la section "🖼️ Convertisseur JPEG/PNG → WebP"
2
Ajuste la qualité (par défaut 85%, recommandé) Plus bas = plus léger mais moins beau. Au-dessus de 95% = ça commence à devenir lourd.
3
Choisis la largeur max (1600px recommandé) Au-delà, les photos sont inutilement grosses pour le web.
4
Glisse tes JPEG/PNG dans la zone Plusieurs à la fois. La conversion est instantanée.
5
Choisis l'action Télécharger : récupère les .webp sur ton PC pour les utiliser plus tard.
Uploader sur GitHub : commit direct dans le dossier media/ que tu indiques (ex: galerie/).
💡 100% local : tes images ne sortent jamais de ton navigateur pendant la conversion.

🔍 Lancer un diagnostic des données

Tout en bas de l'admin, le bouton "Diagnostiquer les données" vérifie automatiquement la santé des fichiers CSV du site.

Il détecte :

  • Colonnes manquantes dans un CSV
  • Doublons (ex: deux fois le même joueur)
  • Dates mal formatées
  • Images référencées qui n'existent pas dans le repo
  • Valeurs invalides dans la colonne "actif"

Le rapport s'affiche dans une fenêtre : vert = OK, orange = à regarder, rouge = à corriger.

💡 À lancer après chaque grosse modif ou de temps en temps, pour éviter qu'un fichier mal formaté casse l'affichage du site.

🆘 En cas de problème

Le site n'affiche pas mes modifs récentes

  • Vérifie que tu as bien cliqué "Publier"/"Enregistrer" jusqu'au bout (toast vert "✓ Publié").
  • Attends 1-2 minutes (propagation Cloudflare).
  • Force le rafraîchissement : Cmd+Shift+R (Mac) ou Ctrl+Shift+R (Windows).
  • Vérifie l'historique GitHub : github.com/theomeurr/New-ChamblyBad/commits/main — ton commit doit y figurer.

Erreur "Token expiré ou invalide"

  • Ton token a expiré ou été révoqué. Recharge la page, on te le redemande.
  • Si tu n'as plus le token, regénère-en un sur github.com/settings/tokens.

Erreur "Conflit : le fichier a été modifié entre-temps"

  • Un autre admin (ou toi-même dans un autre onglet) a modifié le même fichier en parallèle.
  • Clique "Recharger" dans la section concernée et refais tes modifs.

Le formulaire de contact / séance d'essai ne fonctionne pas

  • Vérifie ta connexion internet.
  • Le service Web3Forms est peut-être temporairement indisponible — réessaie dans 5 min.
  • En dernier recours, écris à l'email du club directement.

Je veux annuler une modification

  • Toutes les modifications sont commitées sur GitHub. Pour annuler :
    1. Va sur l'historique des commits.
    2. Trouve le commit à annuler.
    3. Demande à Théo (ou un super-admin) de faire un git revert.
📞 Si tu es vraiment bloqué·e : contacte Théo (créateur du site) ou un autre super-admin. Mieux vaut une question de plus qu'une bêtise irréversible.
